# GuessGame

A simple number guessing game built using HTML, CSS, and JavaScript.  
Users can guess a number, and the game will provide feedback if the guess is too high, too low, or correct.

## Features

- Enter a number and guess
- Receive feedback: Too High / Too Low / Correct
- Simple interactive UI
- Reset the game to play again

## How to Run

1. Clone the repository or download the project.
2. Open `index.html` in your browser.
3. Enter a number and try to guess the correct value.

## Technologies Used

- HTML
- CSS
- JavaScript
- DOM Manipulation
